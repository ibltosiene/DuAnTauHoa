# Train Admin System

Hệ thống quản lý bán vé tàu hỏa dành cho Admin.

## Chức năng

- Dashboard tổng quan
- Quản lý vé
- Quản lý tàu
- Quản lý ga
- Quản lý lịch chạy tàu
- Quản lý khách hàng
- Quản lý thanh toán
- Báo cáo thống kê
- Chính sách giá
- Quản lý hủy vé & hoàn tiền
- Quản lý mã giảm giá

npm install
