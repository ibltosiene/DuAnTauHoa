# 🚆 KLN Train — Portal Điều Phối Viên

## Khởi động nhanh

```bash
# 1. Backend (đã tích hợp, chạy từ thư mục gốc)
cd backend && npm run dev

# 2. Frontend Điều Phối Viên (port 5174)
cd DieuPhoiVien/frontend
npm install
npm run dev

# Truy cập: http://localhost:5174
# Tài khoản: admin@klntrain.vn hoặc nv2@klntrain.vn
```

## Tính năng
- 📊 Dashboard thời gian thực (auto-refresh 60s)
- 🚂 Quản lý chuyến tàu (tìm/lọc/đổi trạng thái)
- 🚃 Quản lý toa theo chuyến (thêm/sửa/xóa/sắp xếp)
- 📋 Ghi nhận sự kiện (chậm, hủy, bảo trì)
- 📅 Quản lý lịch chạy + sinh chuyến hàng loạt

Xem **TaiLieu_HeDieuPhoi.md** để biết thêm chi tiết.
